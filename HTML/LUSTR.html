LUSTR()

Fortran version: 

    SUBROUTINE LUSTR( ASTRING )
        CHARACTER*(*), INTENT(IN   ) :: ASTRING    ! string to make into all caps

NO C version:

Summary:

Upcases (turns into ALL CAPS) and left-justifies (strips leading blanks from) the contents of the Fortran character string ASTRING.

For Fortran-90 declarations and interface checking: 

    USE M3UTILIO

See also LBLANK and UPCASE.

Preconditions:

none

Fortran Usage: 

    ...
    CHARACTER*256  ASTRING
    ...
    ASTRING = '     sOmeThiNg WiERd'
    ...
    CALL LUSTR( ASTRING )   !  now ASTRING is all-caps: 'SOMETHING WIERD'
    ...

C Usage:

Don't. Instead, use standard library routines, e.g. toupper().
